# 臨床文字起こし分析アシスタント (Clinical Transcript Analyzer)

このリポジトリの `index.html` は、精神科外来やカウンセリングの文字起こしデータを AI（Google Gemini API）を用いて分析するためのツールです。

## 主な機能

- **会話パターンの自動抽出**:
    - **会話ラリー**: 医師と患者が短い発言を連続して5往復以上（合計10ターン以上）行っている箇所を特定します。
    - **患者の長い発話（モノローグ）**: 患者が実質的に連続して5文以上（または150文字以上）話し続けている箇所を特定します。
- **多様なファイル形式に対応**:
    - テキストファイル（.txt, .md）
    - Microsoft Word ファイル（.docx）
- **スプレッドシート連携**:
    - 分析結果をコピーし、そのまま Google スプレッドシートなどの表計算ソフトに貼り付けることができます。

## 使い方

1. `index.html` をブラウザで開きます。
2. 右上の **設定（歯車アイコン）** をクリックし、ご自身の [Google AI Studio](https://aistudio.google.com/) で取得した Gemini API キーを入力して保存します。
3. 分析したいテキストを中央のエリアに直接貼り付けるか、「ファイルを開く」からファイルをアップロードします。
4. 「分析開始」ボタンを押すと、AI による解析が始まります。
5. 分析結果は右側のテーブルに表示されます。「スプレッドシートで開く」をクリックすると、データがクリップボードにコピーされ、新しいスプレッドシートが開きます。

## 技術スタック

- **Frontend**: React (via CDN), Tailwind CSS, Lucide React
- **AI**: Google Gemini 2.0 Flash
- **Libraries**: Mammoth.js (Wordファイル解析用)
